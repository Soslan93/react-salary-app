### Prerequisites
You need to install:

```
yarn
```
```
node.js
```
### Installing

Clone Repository

```
Go to project folder
```
Install dependencies
```
yarn
```
Запустить проект командой
```
npm start
```

```
Открыть браузер по адресу
```
```
http://localhost:8080/
```
